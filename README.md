# VolcanoLake
La idea del VolcanoLake es similar a la de FrozenLake. El objetivo del agente será llegar a la meta desde un punto de salida evitando las casillas que penalizan como las de lava o agua y aprovechando tesoros que se puede encontrar a lo largo del camino. El agente aprenderá una política óptima que maximiza las recompensas a través de Q-Learning.
